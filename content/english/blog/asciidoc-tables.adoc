A wide range of table styles are possible by combining the AsciiDoc `frame` and `grid` settings.

.Default (frame plus grid)
|===
|Heading A | Heading B |Heading C

|Cell 1A
|Cell 1B
|Cell 1C

|Cell 2A
|Cell 2B
|Cell 2C

|===

.frame=ends
[frame=ends]
|===

|Cell 1A
|Cell 1B
|Cell 1C

|Cell 2A
|Cell 2B
|Cell 2C

|===

.frame=sides
[frame=sides]
|===
|Heading A | Heading B |Heading C

|Cell 1A
|Cell 1B
|Cell 1C

|Cell 2A
|Cell 2B
|Cell 2C

|===

[frame=none]
|===

|Cell 1A
|Cell 1B
|Cell 1C

|Cell 2A
|Cell 2B
|Cell 2C
